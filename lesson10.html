🔟 استقبال مدخلات من المستخدم في Java
🏠 الرئيسية ⬅ الدرس السابق

📌 ما هي المدخلات؟

المدخلات User Input تعني البيانات التي يقوم المستخدم بإدخالها أثناء تشغيل البرنامج. في Java، نستخدم Scanner لاستقبال المدخلات من لوحة المفاتيح.

🔹 كيفية استقبال مدخلات من المستخدم؟

لاستقبال مدخلات في Java، نستخدم الكود التالي:


import java.util.Scanner; // استيراد الكلاس Scanner

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in); // إنشاء كائن Scanner

        System.out.print("أدخل اسمك: ");
        String name = input.nextLine(); // استقبال النص

        System.out.print("أدخل عمرك: ");
        int age = input.nextInt(); // استقبال رقم صحيح

        System.out.println("مرحبًا " + name + "! عمرك هو " + age + " سنة.");
    }
}

🔹 كيفية استقبال أنواع مختلفة من البيانات؟

يمكن استقبال أنواع مختلفة من البيانات باستخدام Scanner:

نوع البيانات الدالة المستخدمة مثال
نص (String) nextLine() String name = input.nextLine();
عدد صحيح (int) nextInt() int age = input.nextInt();
عدد عشري (double) nextDouble() double price = input.nextDouble();
حرف واحد (char) next().charAt(0) char letter = input.next().charAt(0);
قيمة منطقية (boolean) nextBoolean() boolean isTrue = input.nextBoolean();

🔹 ملاحظة هامة حول استخدام nextLine() بعد nextInt()

عند استخدام nextInt() أو nextDouble() ثم محاولة استخدام nextLine() بعدها، فقد تحدث مشكلة بسبب انتقال السطر الجديد. الحل هو إضافة input.nextLine(); بعد استقبال الرقم. 


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("أدخل عمرك: ");
        int age = input.nextInt();
        input.nextLine(); // لحل مشكلة السطر الجديد

        System.out.print("أدخل اسمك: ");
        String name = input.nextLine();

        System.out.println("مرحبًا " + name + "! عمرك هو " + age + " سنة.");
    }
}

📌 مثال عملي: حساب مجموع رقمين يُدخلهما المستخدم

في هذا المثال، سيقوم المستخدم بإدخال رقمين، والبرنامج سيحسب المجموع.


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("أدخل الرقم الأول: ");
        double num1 = input.nextDouble();

        System.out.print("أدخل الرقم الثاني: ");
        double num2 = input.nextDouble();

        double sum = num1 + num2;
        System.out.println("المجموع: " + sum);
    }
}

  2. ✅ اكتب برنامجًا يطلب من المستخدم إدخال رقمين ثم يحسب الفرق بينهما.
  3. ✅ أنشئ برنامجًا يسأل المستخدم عن درجة حرارته ثم يطبع ما إذا كانت أعلى أو أقل من 37 درجة.
