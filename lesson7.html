7️⃣ الحلقات التكرارية (Loops) في Java
🏠 الرئيسية ➡ الدرس السابق الدرس التالي ⬅

🔄 ما هي الحلقات التكرارية؟

الحلقات التكرارية في Java تُستخدم لتنفيذ كود معين عدة مرات حتى يتحقق شرط معين. هناك ثلاثة أنواع رئيسية:

🔹 1. حلقة for

تُستخدم عندما يكون عدد التكرارات معروفًا.


// طباعة الأرقام من 1 إلى 5
for (int i = 1; i <= 5; i++) {
    System.out.println("الرقم: " + i);
}

🔸 2. حلقة while

تُستخدم عندما لا يكون عدد التكرارات معروفًا مسبقًا.


int count = 1;
while (count <= 5) {
    System.out.println("التكرار: " + count);
    count++;
}

🔻 3. حلقة do-while

تنفذ الكود مرة واحدة على الأقل، حتى لو كان الشرط خاطئًا من البداية.


int number = 1;
do {
    System.out.println("القيمة: " + number);
    number++;
} while (number <= 5);

⚡ تمرين عملي

جرب حل الأسئلة التالية:

  1. ما الفرق بين for و while؟
  2. اكتب كودًا يستخدم for لطباعة الأرقام من 10 إلى 1.
  3. كيف يمكن استخدام while لطباعة الأعداد الزوجية بين 1 و 10؟
