🔍 ما هي أنواع البيانات؟

في Java، يتم استخدام أنواع البيانات لتعريف المتغيرات وتحديد نوع القيم التي يمكنها تخزينها. هناك نوعان أساسيان:

  • الأنواع الأولية (Primitive Types): مثل الأرقام والحروف والقيم المنطقية.
  • الأنواع المرجعية (Reference Types): مثل الكائنات والمصفوفات.

🟢 1. الأنواع الأولية (Primitive Types)

هذه الأنواع هي الأبسط، وتخزن القيم مباشرة في الذاكرة.


int age = 25;       // عدد صحيح
double price = 99.99; // عدد عشري
char grade = 'A';   // حرف واحد
boolean isJavaFun = true; // قيمة منطقية

🔵 2. الأنواع المرجعية (Reference Types)

هذه الأنواع تخزن مراجع للكائنات بدلًا من القيم مباشرة.


String name = "Abdou"; // سلسلة نصية
int[] numbers = {1, 2, 3, 4}; // مصفوفة

⚖️ الفرق بين الأنواع الأولية والمرجعية

الميزة الأنواع الأولية الأنواع المرجعية
طريقة التخزين تخزن القيم مباشرة تخزن مراجع للكائنات
حجم الذاكرة تستخدم ذاكرة أقل تحتاج إلى ذاكرة أكبر
الأداء أسرع في الوصول إلى القيم أبطأ بسبب استخدام المراجع
