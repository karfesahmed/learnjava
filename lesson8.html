8️⃣ المصفوفات (Arrays) في Java
🏠 الرئيسية ➡ الدرس السابق الدرس التالي ⬅

📌 ما هي المصفوفات؟

Arrays هي بنية بيانات تُستخدم لتخزين مجموعة من القيم من نفس النوع داخل متغير واحد.

🔹 كيفية تعريف مصفوفة

يمكنك تعريف مصفوفة في Java باستخدام الكود التالي:


// تعريف مصفوفة تحتوي على 5 أعداد صحيحة
int[] numbers = {10, 20, 30, 40, 50};

// طباعة العنصر الأول
System.out.println(numbers[0]); // النتيجة: 10

🔸 الوصول إلى عناصر المصفوفة

يتم استخدام الفهارس (indices) للوصول إلى عناصر المصفوفة.


int[] ages = {18, 21, 25, 30};
System.out.println("العمر الثاني: " + ages[1]); // النتيجة: 21

🔻 استخدام الحلقات مع المصفوفات

يمكنك استخدام حلقة for للتكرار على جميع عناصر المصفوفة:


String[] names = {"علي", "مريم", "فاطمة"};

for (int i = 0; i < names.length; i++) {
    System.out.println("الاسم: " + names[i]);
}

⚡ تمرين عملي

جرب حل الأسئلة التالية:

  1. ما الفرق بين تعريف مصفوفة من الأعداد ومصفوفة من النصوص؟
  2. قم بإنشاء مصفوفة تخزن أسماء 5 مدن، ثم اطبع جميع القيم باستخدام حلقة for.
  3. اكتب كودًا يتحقق مما إذا كان عدد معين موجودًا داخل مصفوفة أم لا.
