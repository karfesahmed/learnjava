6️⃣ الشروط في Java (if - else if - else)
🏠 الرئيسية ➡ الدرس السابق الدرس التالي ⬅

⚖️ ما هي الشروط في Java؟

تستخدم العبارات الشرطية في Java لاتخاذ قرارات بناءً على القيم المعطاة. إذا تحقق الشرط، يتم تنفيذ كود معين، وإذا لم يتحقق، يمكن تنفيذ كود آخر.

🔹 1. if statement

يتم تنفيذ الكود داخل if فقط إذا كان الشرط صحيحًا (true).


int age = 20;
if (age >= 18) {
    System.out.println("أنت بالغ.");
}

🔸 2. if - else

إذا لم يتحقق شرط if، يتم تنفيذ الجزء داخل else.


int age = 16;
if (age >= 18) {
    System.out.println("يمكنك القيادة.");
} else {
    System.out.println("لا يمكنك القيادة بعد.");
}

🔷 3. if - else if - else

تستخدم else if عندما يكون لدينا عدة احتمالات.


int age = 14;
if (age >= 18) {
    System.out.println("يمكنك القيادة.");
} else if (age >= 16) {
    System.out.println("يمكنك القيادة برخصة مؤقتة.");
} else {
    System.out.println("عليك الانتظار حتى تكبر.");
}

✍️ تمرين

قم بإنشاء متغير age يمثل عمر المستخدم، ثم استخدم الشروط للتحقق من الفئات العمرية التالية:

  1. إذا كان العمر أقل من 13، اطبع: "You are a child."
  2. إذا كان بين 13 و 19، اطبع: "You are a teenager."
  3. إذا كان 20 أو أكثر، اطبع: "You are an adult."
