📦 المتغيرات في Java

المتغير هو مكان في الذاكرة يُستخدم لتخزين البيانات. يمكن أن يكون عددًا، نصًا، أو حتى كائنًا.

📝 1. كيفية تعريف متغير

لتعريف متغير في Java، يجب تحديد نوعه أولًا، ثم إعطاؤه اسمًا، وأخيرًا إسناد قيمة إليه.


int age = 19;       // متغير لتخزين العمر
String name = "Abdou"; // متغير لتخزين الاسم
  • int: يستخدم لتخزين الأعداد الصحيحة.
  • double: يستخدم لتخزين الأعداد العشرية.
  • String: يستخدم لتخزين النصوص.
  • boolean: يستخدم لتخزين القيم المنطقية (true أو false).

🔍 2. طباعة القيم المخزنة

يمكننا طباعة القيم باستخدام System.out.println : 


System.out.println("Welcome " + name + "! You have " + age + " years old.");

✍️ تمرين

قم بإنشاء متغيرين كما يلي:

  1. متغير name يحتوي على اسمك.
  2. متغير age يحتوي على عمرك.
  3. قم بطباعة رسالة ترحيب مثل: Welcome Abdou! You have 19 years old.
