9️⃣ الدوال (Functions) في Java
🏠 الرئيسية ➡ الدرس السابق الدرس التالي ⬅

📌 ما هي الدوال؟

الدوال Functions هي كتل من الكود يتم تعريفها لتنفيذ مهمة معينة عند استدعائها. تساعد الدوال في جعل الكود أكثر تنظيمًا، كما تقلل من التكرار.

🔹 لماذا نستخدم الدوال؟

  • ✅ لتقليل تكرار الكود.
  • ✅ لتقسيم البرنامج إلى أجزاء أصغر يسهل إدارتها.
  • ✅ لجعل الكود أكثر قابلية للفهم والتعديل.

🔸 كيفية تعريف دالة في Java

تعريف دالة يتكون من الأجزاء التالية:

  • public: تعني أن الدالة يمكن استدعاؤها من أي مكان في البرنامج.
  • static: تعني أن الدالة تابعة للكلاس Class وليس للكائنات.
  • void: تعني أن الدالة لا ترجع أي قيمة.
  • اسم الدالة: يجب أن يكون اسمًا يصف وظيفتها.
  • المعاملات (Parameters): القيم التي تستقبلها الدالة (يمكن أن تكون فارغة).
  • الجسم: وهو الكود الذي يتم تنفيذه عند استدعاء الدالة.

📌 مثال على دالة بسيطة:

هذه الدالة تطبع رسالة ترحيبية.


public class Main {
    public static void sayHello() {
        System.out.println("مرحبًا بالجميع!");
    }

    public static void main(String[] args) {
        sayHello(); // استدعاء الدالة
    }
}

🔹 الدوال التي تُرجع قيمة

بعض الدوال تُرجع قيمة باستخدام return:


public class Main {
    public static int sum(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        int result = sum(5, 3);
        System.out.println("المجموع: " + result); // النتيجة: 8
    }
}

🔹 الدوال التي تستقبل معاملات (Parameters)

يمكن للدالة استقبال بيانات عند استدعائها:


public class Main {
    public static void greet(String name) {
        System.out.println("مرحبًا، " + name + "!");
    }

    public static void main(String[] args) {
        greet("أحمد");
        greet("سارة");
    }
}

📌 مثال عملي: حساب السعر بعد الضريبة

دالة لحساب السعر بعد إضافة ضريبة 15%:


public class Main {
    public static double calculatePrice(double price) {
        double tax = price * 0.15;
        return price + tax;
    }

    public static void main(String[] args) {
        double finalPrice = calculatePrice(100);
        System.out.println("السعر النهائي: " + finalPrice); // 115.0
    }
}

⚡ تمرين عملي

جرب حل الأسئلة التالية:

  1. ✅ قم بإنشاء دالة ترجع أكبر عدد بين رقمين.
  2. ✅ اكتب دالة تستقبل اسم شخص وعمره ثم تطبع رسالة ترحيبية.
  3. ✅ أنشئ دالة تحسب عدد الأحرف في كلمة معينة وترجع النتيجة.
