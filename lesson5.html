🧮 العمليات الحسابية في Java

تدعم Java مجموعة من العمليات الحسابية مثل الجمع، الطرح، الضرب، القسمة، وباقي القسمة.

➕ 1. العمليات الحسابية الأساسية

فيما يلي بعض العمليات الأساسية في Java:


int a = 10;
int b = 5;

int sum = a + b;      // جمع
int difference = a - b; // طرح
int product = a * b;  // ضرب
int quotient = a / b; // قسمة
int remainder = a % b; // باقي القسمة
  • الجمع (+): لحساب مجموع رقمين.
  • الطرح (-): لحساب الفرق بين رقمين.
  • الضرب (*): لحساب ناتج ضرب رقمين.
  • القسمة (/): لحساب ناتج القسمة (قسمة صحيحة للأعداد الصحيحة).
  • باقي القسمة (%): لحساب باقي القسمة بين رقمين.

🔄 2. العمليات المختصرة (التخصيص المركب)

توفر Java طرقًا مختصرة لإجراء العمليات الحسابية باستخدام معاملات التخصيص المركب.


int x = 10;

x += 5; // x = x + 5
x -= 3; // x = x - 3
x *= 2; // x = x * 2
x /= 4; // x = x / 4
x %= 3; // x = x % 3
  • +=: يضيف القيمة إلى المتغير.
  • -=: يطرح القيمة من المتغير.
  • *=: يضرب المتغير بالقيمة.
  • /=: يقسم المتغير على القيمة.
  • %=: يأخذ باقي القسمة للمتغير.

📈 3. العمليات على الأعداد العشرية

عند التعامل مع الأعداد العشرية، نستخدم double بدلاً من int لضمان دقة العمليات.


double num1 = 7.5;
double num2 = 2.5;

double result = num1 / num2; // نتيجة القسمة العشرية
